```bash

flutter run
```

### **For Android Emulator:**
```bash
cd mobile
flutter emulators --launch Pixel
flutter run -d android
```

### **Important:** 
- ✅ Add your Google Maps API key to `mobile/.env`
- ✅ Add the same key to `mobile/android/local.properties`
